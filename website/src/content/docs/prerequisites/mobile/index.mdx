### Anforderungen an Ihr mobiles Endgerät

Die Urban Feed Monitor-App steht sowohl für Endgeräte mit <span  class="font-semibold">Android-Betriebssystem</span> als auch für <span  class="font-semibold">Apple-Geräte</span> mit iOS zur Verfügung.

* Für Endgeräte mit dem Betriebssystem Android ist eine <span  class="font-semibold">Android-Version von 7.0 oder höher</span> erforderlich.

* Für Apple-Endgeräte ist eine <span class="font-semibold">iOS-Version von 15.6</span> oder höher erforderlich.
