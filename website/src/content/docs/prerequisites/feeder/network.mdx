## Weiteres Vorgehen

* Verbinden Sie den Ist Urban-Router Ihres Fütterungssystems mit dem Internet falls nicht bereits geschehen.
* Ist/sind ihr(e) Automat(en) noch nicht mit einem Urban Router ausgestattet, so beauftragen Sie ihren Vertriebspartner damit, ihr Fütterungssystem mit einem Urban-Router auszustatten.
