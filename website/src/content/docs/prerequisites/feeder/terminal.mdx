### Anforderungen an den Touch-Terminal

Für die Installation und das Setup der Urban-Cloud ist ein Terminal der  <span class="font-bold">Generation 2 </span> (verkauft ab 2020) oder höher zwingend erforderlich.

### Ermittlung der Generation

So prüfen Sie, welche Generation des Touch-Terminals Sie haben:


* Wechseln Sie zur Startseite des Automaten, klicken Sie dazu ggf. auf die oberste `Home`-Schaltfläche <span class="text-2xl">➀</span> <Home width={20} class="inline ml-2 mb-0" /> in der Navigationsleiste im linken Bildschirmbereich.
* Klicken Sie im Startbildschirm auf die `Hilfe`-Schaltfläche <span class="text-2xl">➁</span> <QuestionMark width={20} align="top" class="inline ml-2 mb-0" alt="Icon Hilfemenü" /> im rechten oberen Bildschirmbereich.
* Die installierte Hardware-Generation <span class="text-2xl">➂</span> wird jetzt links neben der Schaltfläche zur Sprachumschaltung neben "Hardware" angezeigt.
