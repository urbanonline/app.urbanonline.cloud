### Anforderungen an das Modell des Tränkeautomaten

### Weiteres Vorgehen

* Handelt es sich bei ihrem Automaten nicht um eine Alma Pro, so können Sie die App leider nicht ausführen. Erwägen Sie eine Ersatzinvestition für ihren Tränkeautomaten und erwerben Sie einen Tränkeautomten der Modellreihe *Alm Pro*.
