### Anforderungen an die Firmwareversion

Für die Installation und den Setup der Urban-Cloud ist eine Firmwareversion <span class="font-bold">25-04-V08</span> oder höher zwingend erforderlich.

### Ermittlung der Firmwareversion

So prüfen Sie, welche Firmwareversion auf ihrem Automaten installiert ist:

* Wechseln Sie zur Startseite des Automaten, klicken Sie dazu ggf. auf die oberste `Home`-Schaltfläche <span class="text-2xl">➀</span> <Home width={20} class="inline ml-2 mb-0" /> in der Navigationsleiste im linken Bildschirmbereich.
* Klicken Sie im Startbildschirm auf die `Hilfe`-Schaltfläche <span class="text-2xl">➁</span> <QuestionMark width={20} align="top" class="inline ml-2 mb-0" alt="Icon Hilfemenü" /> im rechten oberen Bildschirmbereich.
* Die Versionsnummer der installierten Automatenfirmware <span class="text-2xl">➂</span> wird jetzt links neben der Schaltfläche zur Sprachumschaltung angezeigt.

### Weiteres Vorgehen

* Ist die Firmwareversion ihres Automaten niedriger als 25-04-V08, so nehmen Sie zunächst ein [Firmwareupdate](../../../guide/service/update-feeder) ihres Automaten vor.

* Ist auf Ihrem Automaten bereits die Firmwareversion 25-04-V08 (oder höher) installiert und sind auch die alle anderen [Anforderungen](../) erfüllt, so fahren Sie direkt fort mit Schritt <span class="text-2xl">③</span> [Cloud Setup](../../../guide/service/setup-cloud).
