Pour commencer l'installation, vous devez d'abord accéder à l'écran d'installation du service Urban Cloud :

### Accès à l'écran d'installation

<Tabs>
<TabItem label="Version courte">
Cliquez, comme indiqué sur la [capture d'écran 1](#screenshot1) ci-dessous, successivement sur les boutons <span class="text-2xl">➀</span>&nbsp;à&nbsp;<span class="text-2xl">⑤</span>.
</TabItem>
<TabItem label="Description détaillée">
1. Dans la liste latérale gauche du terminal tactile, cliquez sur le bouton du bas <Gear width={20} class="inline ml-2 mb-0" /> `Paramètres` <span class="text-2xl">➀</span>. Si vous êtes dans le menu principal, où aucune barre latérale n'est visible, appuyez sur le bouton <Feeder width={20} class="inline ml-2 mb-0" /> `Distributeur` pour afficher la barre latérale.
1. Dans la barre d'outils inférieure du terminal tactile, cliquez sur le deuxième bouton en partant de la gauche <TouchGear width={30} class="inline ml-2 mb-0" /> `Paramètres du terminal tactile` <span class="text-2xl">➁</span>.
1. Une série d'éléments de menu apparaît maintenant dans la moitié gauche de la zone centrale de l'écran. Cliquez sur le bouton du bas `Accessoires` <span class="text-2xl">➂</span>.
1. Un (ou éventuellement plusieurs) élément(s) de menu apparaît maintenant dans la moitié droite de la zone centrale de l'écran. Cliquez sur le bouton du bas `Service Cloud Urbain` <span style="font-size: 140%" id="Navigation_Digit_4">➃</span>. Le bouton passe maintenant dans la moitié gauche de l'écran.
1. En bas, dans la moitié droite de l'écran, le bouton <Edit width={20} class="inline ml-2 mb-0" /> `Ouvrir réglages` <span style="font-size: 140%" id="Navigation_Digit_5">➄</span> s'affiche maintenant. Cliquez sur ce bouton ; une fenêtre contextuelle s'ouvre alors, listant les [étapes](../setup-cloud) de configuration du service Urban Cloud.
</TabItem>
</Tabs>
