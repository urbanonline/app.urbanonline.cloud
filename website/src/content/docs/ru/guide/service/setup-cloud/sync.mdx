### <a id="installation-sync-service"></a>Установка службы синхронизации

Сначала необходимо установить программный пакет службы синхронизации; этот пакет загружается через интернет из Urban Cloud. 

  <Aside type="caution" title="Версия программного пакета" icon="information">Номер версии устанавливаемого программного пакета отображается в раскрывающемся меню с красной рамкой под заголовком `Доступные версии` (буква&nbsp;<span class="text-2xl">Ⓐ</span>); этот номер версии имеет вид `prod - n.n.n - ГГГГ-ММ-ДД ЧЧ:MM:СС`.</Aside>

- Убедитесь, что в раскрывающемся меню <span class="text-2xl">Ⓐ</span> `Доступные версии` выбрана самая новая версия программного пакета. Для этого нажмите кнопку с красной рамкой (цифра <span class="text-2xl">➀</span>), чтобы открыть список версий раскрывающегося меню. Затем выберите в отображённом списке самый новый по времени номер версии.
- Нажмите кнопку `Установить` (цифра <span class="text-2xl">➁</span>), чтобы начать установку службы синхронизации.
- После непродолжительного времени установки содержимое всплывающего окна изменится, и появится большая зелёная галочка <span class="p-0.5 bg-green-600 text-white">✓</span>, сообщающая об успешной установке. Подтвердите это сообщение об успехе.

### Запуск службы синхронизации

Чтобы только что успешно установленная служба синхронизации начала работать, её необходимо запустить ещё одним шагом на том же экране ([скриншот 2](#screenshot2)).

- Нажмите кнопку `Начать` (цифра <span class="text-2xl">➂</span>) под заголовком <span class="text-2xl">Ⓑ</span> `управление сервисом`, чтобы запустить службу синхронизации.
- После непродолжительного времени запуска содержимое всплывающего окна изменится, и появится большая зелёная галочка <span class="p-0.5 bg-green-600 text-white">✓</span>, сообщающая об успешном запуске службы. Подтвердите это сообщение об успехе.
- После успешной настройки службы нажмите кнопку с символом <span class="p-0.5 bg-green-600 text-white">✓</span> (цифра&nbsp;<span class="text-2xl">➃</span>), чтобы покинуть экран установки.
- Индикатор состояния ([скриншот 1](#screenshot1), цифра <span class="text-2xl">➀</span>) слева от пункта списка `Data transmission active` теперь должен гореть <span class="p-0.5 bg-green-600 text-white">зелёным</span>, сигнализируя об успешной установке и запуске службы синхронизации.

На этом установка Urban Cloud на кормовом автомате Alma Pro успешно завершена.
