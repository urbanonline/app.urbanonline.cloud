На скриншоте 1 перечислены четыре шага установки, которые необходимо выполнить последовательно в ходе настройки службы Urban Cloud. Нажимайте поочерёдно на кнопки &nbsp;<Gear width={20} class="inline mb-0" />, расположенные справа от каждого пункта списка, и выполняйте на открывшемся экране необходимые действия.

После каждого шага один из четырёх индикаторов состояния слева, изначально красных, должен загореться зелёным. После успешной настройки четыре зелёных индикатора сигнализируют о том, что облако настроено корректно и работает.
