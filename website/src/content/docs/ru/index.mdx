## Возможности приложения

<CardGrid stagger>
	<Card title="Иерархическая структура помещений" icon="farcaster" >
		Создавайте `Помещения` и привязывайте к ним `Автоматы`.
	</Card>
	<Card title="Фильтры: помещения, автоматы" icon="question-circle">
		Просматривайте животных из всех помещений, из конкретного помещения или с конкретного автомата.
	</Card>
	<Card title="Поиск животных" icon="magnifier">
		`Найдите` нужное животное в списке стада по его `номеру`.
	</Card>
	<Card title="Информация по отдельному животному" icon="information">
		Просматривайте животных по очереди и открывайте информацию о животном или сводку по кормлению.
	</Card>
	<Card title="Рабочее состояние автоматов" icon="analytics">
		Просматривайте `рабочее состояние` всех автоматов, включая `сообщения об ошибках`.
	</Card>
	<Card title="Push-уведомление при ошибке автомата" icon="comment-alt">
		При возникновении ошибки на автомате вы получите `push-уведомление`.
	</Card>
	<Card title="Гибкое управление пользователями" icon="setting">
		Создавайте сколько угодно `дополнительных пользователей` и приглашайте их пользоваться приложением вместе с вами.
	</Card>
