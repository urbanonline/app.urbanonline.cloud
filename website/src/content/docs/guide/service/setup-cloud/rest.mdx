## REST-Schnitstelle aktivieren

Tippen Sie auf das Zahnrad rechts neben "REST-API aktiv", um das Menü zu öffnen.

<figure>
<div class="border border-gray-300 rounded-xs p-4">
<Image src={ActivationREST}  alt="Installation Urban-Cloud, Aktivierung REST-Schnitstelle" title="Aktivierung REST-Schnitstelle" />
  </div>
  <figcaption class="text-sm mt-2">
    <span class="">Screenshot 2:</span> Cloud-Setup, Aktivierung der REST-Schnitstelle
  </figcaption>
</figure>

Schieben Sie den Regler nach rechts auf "JA" und bestätigen Sie mit einem Tippen auf den grünen Haken.

Der Punkt links neben "REST-API aktiv" ist nun grün.
