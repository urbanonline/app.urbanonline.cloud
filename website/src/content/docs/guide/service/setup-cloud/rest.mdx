## Status der REST-Schnittstelle prüfen

Im dritten Schritt der Installation muss geprüft werden, ob die REST-Schnittstelle des Touch-Terminal aktiviert ist und Anfragen entgegennehmen kann. Die Farbe der kreisförmigen Statusanzeige (Ziffer&nbsp;<span class="text-2xl">➀</span>) links neben dem dritten Listeneintrag `REST-API aktiv` im Installationsbildschirm gibt darüber Auskunft:

  <div class="border border-gray-300 rounded-xs p-4">
    <Image src={REST_API}  alt="Installation Urban-Cloud, Rest-Schnittstelle" title="Rest-Schnittstelle" />
  </div>
  <figcaption class="text-sm mt-2">
    <span class="">Screenshot 1:</span> Cloud-Setup, Aktivierungsstatus der REST-Schnittstelle
  </figcaption>
</figure>

- Leuchtet die Statusanzeige <span class="text-2xl">➀&nbsp;</span><span class="p-0.5 bg-green-600 text-white">grün</span>, so wurde die REST-Schnittstelle bereits aktiviert. Alles ist in Ordnung, Sie können im letzten Installationsschritt die [Datenübertragung in die Cloud](../sync) aktivieren.

- Leuchtet die Statusanzeige <span class="p-0.5 bg-red-500 text-white">rot</span>, so wurde die REST-Schnittstelle noch nicht aktiviert, es besteht also Handlungsbedarf. 

    - Tippen Sie auf die rechts neben dem Listeneintrag `REST-API aktiv` befindliche Schaltfläche&nbsp;<Gear width={20} class="inline mb-0" /> (Ziffer <span class="text-2xl">➁</span>) um den Aktivierungsbildschirm für die REST-Schnittstelle aufzurufen ([Screenshot 2](#screenshot2)).

    - Aktivieren Sie die REST-Schnittstelle gemäß den Anweisungen im nächsten Abschnitt.

## <a id="activate-rest-api"></a> REST-Schnittstelle aktivieren

Die Aktivierung der Rest-Schnittstelle erfolgt in folgendem Bildschirm:

<figure id="screenshot2">
<div class="border border-gray-300 rounded-xs p-4">
<Image src={ActivationREST}  alt="Installation Urban-Cloud, Aktivierung REST-Schnittstelle" title="Aktivierung REST-Schnittstelle" />
  </div>
  <figcaption class="text-sm mt-2">
    <span class="">Screenshot 2:</span> Cloud-Setup, Aktivierung der REST-Schnittstelle
  </figcaption>
</figure>

- Tippen Sie auf die Option `JA` des Schiebereglers (Ziffer <span class="text-2xl">➀</span>).

- Tippen Sie auf die Schaltfläche <span class="p-0.5 bg-green-600 text-white">✓</span> `OK` (Ziffer&nbsp;<span class="text-2xl">➁</span>) damit die Änderung wirksam wird.

- Der Inhalt des Popup-Fensters wechselt jetzt, es erscheint ein großes grünes Häkchen <span class="p-0.5 bg-green-600 text-white">✓</span>, welches den Erfolg der Aktivierung mitteilt.

- Quittieren Sie dieses Erfolgsmeldung.

- Die Statusanzeige ([Screenshot 1](#screenshot1), Ziffer <span class="text-2xl">➀</span>) links neben dem Listeneintrag `REST-API aktiv` sollte jetzt <span class="p-0.5 bg-green-600 text-white">grün</span> leuchten und damit signalisieren, dass die REST-Schnittstelle erfolgreich aktiviert wurde.

Aktivieren Sie jetzt im letzten Installationsschritt die [Datenübertragung in die Cloud](../sync).
