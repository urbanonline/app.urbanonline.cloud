## Erreichbarkeit des Cloud-Servers prüfen

Zu Beginn der Installation muss geprüft werden, ob der Urban-Cloud-Server über das Internet erreicht werden kann. Die Farbe der kreisförmigen Statusanzeige <span class="text-2xl">➀</span> links neben dem obersten Punkt `Verbindung zum Server` im Installationsbildschirm gibt darüber Auskuft:

<figure>
<div class="border border-gray-300 rounded-xs p-4">
<Image src={ConnectionServer}  alt="Installation Urban-Cloud, Prüfung Netzwerkverbindung" />
  </div>
  <figcaption class="text-sm mt-2">
    <span class="">Screenshot 1:</span> Cloud-Setup, Status zur Erreichbarkeit des Cloud-Servers
  </figcaption>
</figure>

- Leuchtet die Statusanzeige <span class="p-0.5 bg-green-600 text-white">grün</span>, so kann der Cloud-Server erreicht werden. Alles ist in Ordnung, Sie können diese Seite verlassen und mit der [Registrierung des Terminals](../registration) fortfahren.

- Leuchtet die Statusanzeige <span class="p-0.5 bg-red-500 text-white">rot</span>, so kann der Cloud-Server **nicht** erreicht werden, es besteht also Handlungsbedarf. Lesen Sie weiter und befolgen Sie die nachstehend aufgelisteten Handlungsschritte.

## Details zur Erreichbarkeit einsehen

- Tippen Sie auf die rechts befindliche Schaltfläche <span class="text-2xl">➁</span> &nbsp; <Gear width={20} class="inline ml-2 mb-0" /> (Screenshot 1) um Details zur Netzwerkverbindung einzusehen. Es wird zu einem Bildschirm umgeschaltet (Screenshot 2), der Auskunft darüber gibt, ob der Urban-Netzwerkrouter <span class="text-2xl">➀</span>, das Internet <span class="text-2xl">➁</span> und letzlich der Urban Cloud-Server <span class="text-2xl">③</span> ereicht werden kann :

<figure>
<div class="border border-gray-300 rounded-xs p-4">
<Image src={NetworkDetails}  alt="Schritt 1" />
  </div>
  <figcaption class="text-sm mt-2">
    <span class="">Screenshot 2:</span> Alma Pro, Cloud-Setup, Details zur Erreichbarkeit des Cloud-Servers
  </figcaption>
</figure>

Eine <span class="p-0.5 bg-green-600 text-white">grüne</span> Statusanzeige signalisiert jeweils, dass die Komponente erreicht werden kann, eine <span class="p-0.5 bg-red-500 text-white">rote</span> Statusanzeige zeigt Nichterreichbarkeit an. Nutzen Sie die aufgelisteten Details bei ihrer Fehleranalyse. Stellen Sie nacheinander durch geeignete Verkabelung und Funktionstests der Netzwerkkomponenten sicher, dass alle drei Statusanzeigen <span class="p-0.5 bg-green-600 text-white">grün</span> aufleuchten. Ist dies der Fall, so haben Sie Verbindungsproblem erfolgreich gelöst und sie können mit dem nächsten Schritt [`Registrierung des Terminals`](../registration) fortfahren.
