## Anleitung Schritt 1
Tippen Sie auf das Zahnrad, um die Einstellungen zu öffnen, dann auf "Zubehör" und "Urban Cloud Service".
<Image src={DE_Step1} alt="Schritt 1" width={500} />
Tippen Sie unten rechts auf "Einstellungen öffnen". 
<Image src={DE_Step2} alt="Schritt 2" width={500} />

Die farbigen Punkte zeigen den Status an und sollten alle grün sein. Sind sie rot oder orange, besteht Handlungsbedarf. Tippen Sie auf das Zahnrad rechts neben "Verbindung zum Server", um das Menü zu öffnen.
<Image src={DE_Step3} alt="Schritt 3" width={500} />
Es gibt ein Problem mit der Verbindung zum Server.
<Image src={DE_Step3a} alt="Schritt 3a" width={500} />
Sie haben das Problem erfolgreich gelöst und der Punkt links neben "Verbindung zum Server" ist nun grün.
<Image src={VerbindungZumServer} alt="VerbindungZumServer" width={500} />
