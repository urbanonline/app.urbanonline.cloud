## Erreichbarkeit des Cloud-Servers prüfen

Zu Beginn der Installation muss geprüft werden, ob der Urban-Cloud-Server über das Internet erreicht werden kann. Die Farbe der kreisförmigen Statusanzeige (Ziffer&nbsp;<span class="text-2xl">➀</span>) links neben dem obersten Listeneintrag `Verbindung zum Server` im Installationsbildschirm gibt darüber Auskunft:

<figure>
<div class="border border-gray-300 rounded-xs p-4">
<Image src={ConnectionServer}  alt="Installation Urban-Cloud, Prüfung Netzwerkverbindung" title="Prüfung Netzwerkverbindung" />
  </div>
  <figcaption class="text-sm mt-2">
    <span class="">Screenshot 1:</span> Cloud-Setup, Status zur Erreichbarkeit des Cloud-Servers
  </figcaption>
</figure>

- Leuchtet die Statusanzeige <span class="text-2xl">➀&nbsp;</span><span class="p-0.5 bg-green-600 text-white">grün</span>, so kann der Cloud-Server erreicht werden. Alles ist in Ordnung, Sie können im nächsten Installationsschritt mit der [Registrierung des Terminals](../registration) fortfahren.

- Leuchtet die Statusanzeige <span class="p-0.5 bg-red-500 text-white">rot</span>, so kann der Cloud-Server **nicht** erreicht werden, es besteht also Handlungsbedarf. Tippen Sie auf die rechts rechts neben dem Listeneintrag befindliche Schaltfläche&nbsp;<Gear width={20} class="inline ml-2 mb-0" /> (Ziffer <span class="text-2xl">➁</span>) zu einem Detail-Bildschirm weiterzuschalten ([Screenshot 2](#screenshot2)) dessen Inhalte Sie bei der Fehlersuche unterstützen

## <a id="view-connectivity-details"></a> Prüfung der Erreichbarkeit von Netzwerkteilnehmern

Der folgende [Screenshot](#screenshot2) gibt Auskunft darüber, ob nacheinander jeweils der Urban-Netzwerkrouter&nbsp;<span class="text-2xl">➀</span>, das Internet&nbsp;<span class="text-2xl">➁</span> und letzlich der Urban Cloud-Server&nbsp;<span class="text-2xl">③</span> ereicht werden kann :

<figure id="screenshot2">
<div class="border border-gray-300 rounded-xs p-4">
<Image src={NetworkDetails}  alt="Installation Urban-Cloud, Status Netzwerkverbindung" title="Status Netzwerkverbindung" />
  </div>
  <figcaption class="text-sm mt-2">
    <span class="">Screenshot 2:</span> Alma Pro, Cloud-Setup, Details zur Erreichbarkeit des Cloud-Servers
  </figcaption>
</figure>

- Eine <span class="p-0.5 bg-green-600 text-white">grüne</span> Statusanzeige signalisiert jeweils, dass die Komponente erreicht werden kann, eine <span class="p-0.5 bg-red-500 text-white">rote</span> Statusanzeige zeigt Nichterreichbarkeit an. Nutzen Sie die aufgelisteten Details bei ihrer Fehleranalyse.

- Stellen Sie nacheinander durch geeignete Verkabelung und Funktionstests der Netzwerkkomponenten sicher, dass alle drei Statusanzeigen <span class="p-0.5 bg-green-600 text-white">grün</span> aufleuchten.

  <Aside type="caution" title="Hinweis" >Ziehen Sie ggf. denjenigen Netzwerktechniker zu Rate, der für das Netzwerk und die Internetverbindung auf dem Betrieb zuständig hat.</Aside>

- Leuchten alle drei Statusanzeigen <span class="p-0.5 bg-green-600 text-white">grün</span> auf, so haben Sie Verbindungsproblem erfolgreich gelöst und sie können mit dem nächsten Schritt [`Registrierung des Terminals`](../registration) fortfahren.
