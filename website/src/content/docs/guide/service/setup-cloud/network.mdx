## Einstellungsmenü `Urban Cloud Service` aufrufen

Die farbigen Punkte zeigen den Status an und sollten alle grün sein. Sind sie rot oder orange, besteht Handlungsbedarf. Tippen Sie auf das Zahnrad rechts neben "Verbindung zum Server", um das Menü zu öffnen.
