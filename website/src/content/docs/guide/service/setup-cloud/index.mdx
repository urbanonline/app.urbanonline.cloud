Screenshot 1 listet die vier Installationsschritte auf, die im Zuge des Setup für den Urban Cloud- Service nacheinander durchlaufen werden müssen. Klicken Sie nacheinander jeweils auf die rechts neben dem Listenelement befindlichen Schaltflächen &nbsp;<Gear width={20} class="inline mb-0" /> und nehmen Sie in dem nun aufgerufenen Bildschirm die jeweils notwendigen Aktionen vor.

Nach jedem Schritt muss eine weitere der vier linken, anfänglich <span class="p-0.5 bg-red-600 text-white">roten</span> Statusanzeigen <span class="p-0.5 bg-green-600 text-white">grün</span> aufleuchten. Nach dem erfolgreichen Setup signalisieren vier <span class="p-0.5 bg-green-600 text-white">grüne</span> Anzeigen, dass die CLoud funktionsfähig eingerichtet wurde.
