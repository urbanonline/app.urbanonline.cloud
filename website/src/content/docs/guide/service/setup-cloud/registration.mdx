## Registrierungsstatus des Touch-Terminal prüfen

Im zweiten Schritt der Installation muss geprüft werden, ob das Touch-Terminal bereits bei Urban Cloud-Server registriert ist, nur dann können erfolgreich Daten an die Urban-Cloud übermittelt werden. Die Farbe der kreisförmigen Statusanzeige (Ziffer&nbsp;<span class="text-2xl">➀</span>) links neben dem dritten Listeneintrag `Registrierung am Server` im Installationsbildschirm gibt darüber Auskunft:

  <div class="border border-gray-300 rounded-xs p-4">
    <Image src={Registrierung}  alt="Installation Urban-Cloud, Registrierungsstatus des Touch-Terminals" title="Registrierung Touch-Terminal" />
  </div>
  <figcaption class="text-sm mt-2">
    <span class="">Screenshot 1:</span> Cloud-Setup, Registrierungsstatus des Touch-Terminals
  </figcaption>
</figure>

- Leuchtet die Statusanzeige <span class="text-2xl">➀&nbsp;</span><span class="p-0.5 bg-green-600 text-white">grün</span>, so wurde die REST-Schnittstelle bereits aktiviert. Alles ist in Ordnung, Sie können im dritten Installationsschritt die [REST-Schnittstelle](../rest) aktivieren.

- Leuchtet die Statusanzeige <span class="p-0.5 bg-red-500 text-white">rot</span>, so wurde das Terminal noch nicht beim Cloud-Server registriert, es besteht also Handlungsbedarf. 

    - Tippen Sie auf die rechts neben dem Listeneintrag `Registrierung am Server` befindliche Schaltfläche&nbsp;<Gear width={20} class="inline mb-0" /> (Ziffer <span class="text-2xl">➁</span>) um den Bildschirm für die Registrierung des Touch-Terminals aufzurufen ([Screenshot 2](#screenshot2)).

    - Folgen Sie den Anweisungen im nächsten Abschnitt und registrieren Sie das Touch Terminal beim Urban Cloud-Server.

## <a id="registration-terminal"></a>Touch-Terminal beim Cloud-Server registrieren

:::caution[Benutzerkonto in der App zwingend erforderlich]
Eine erfolgreiche Registrierung des Terminals am Cloud-Server setzt voraus, dass in der Urban Feed Monitor App ein [**Benutzerkonto**](/guide/user/01-registrierung/) erstellt wurde.
:::

- [Melden](/guide/user/02-einloggen/) Sie sich mit ihrem Benutzerkonto in der Urban Feed Monitor-App an.

- [Registrieren](/guide/user/04-terminal-hinzufuegen/) Sie das Touch-Terminal am Urban-Cloud-Server, indem Sie den QR-Code auf dem Bildschirm mit dem mobilen Endgerät einscannen. 

<figure id="screenshot1">
  <div class="border border-gray-300 rounded-xs p-4">
    <Image src={QRCode}  alt="Installation Urban-Cloud, QR-Code zur Registrierung des Touch-Terminals" title="QR-Code Registrierung Touch" />
  </div>
  <figcaption class="text-sm mt-2">
    <span class="">Screenshot 2:</span> QR-Code mit [UUID](https://de.wikipedia.org/wiki/Universally_Unique_Identifier) zur Registrierung des Touch-Terminals
  </figcaption>
</figure>

- Die Statusanzeige ([Screenshot 1](#screenshot1), Ziffer <span class="text-2xl">➀</span>) links neben dem Listeneintrag `Registrierung am Server` sollte jetzt <span class="p-0.5 bg-green-600 text-white">grün</span> leuchten und damit signalisieren, dass das Touch-Terminal erfolgreich am Urban-Clouserver registriert wurde.

Aktivieren Sie jetzt im nächsten Installationsschritt die lokale [REST-Schnittstelle](../rest) des Touch-Terminals.
